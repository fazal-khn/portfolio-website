> 🌐 **Live Portfolio:** [fazal-khn.github.io/portfolio-website](https://fazal-khn.github.io/portfolio-website)

A premium, pixel-perfect developer portfolio built with **Vanilla HTML, CSS, and JavaScript** — no frameworks, no bloat. Fast, responsive, and visually stunning with glassmorphic UI, animated gradients, and an interactive project showcase.

---

## ✨ Design Highlights

- **Dark Glassmorphism UI** — frosted-glass cards with multi-layered depth effects
- **Animated Gradient Orbs** — three ambient glow blobs that drift across the background
- **Magnetic Card Glow** — project cards respond to mouse position with a radial spotlight effect
- **Smooth Scroll & Intersection Observers** — sections animate into view as you scroll
- **Category Filter Tabs** — toggle between Web, Mobile, and All Projects with smooth transitions
- **Collapsible Detail Panels** — technical implementation notes expand on demand
- **Responsive Drawer Nav** — mobile-friendly hamburger menu with an animated slide drawer

## 🛠️ Tech Used

- **Structure**: Semantic HTML5 (single-page, multi-section layout)
- **Styling**: Pure Vanilla CSS with CSS Custom Properties, Grid, Flexbox, and Clamp-based fluid typography
- **Interactivity**: Vanilla JavaScript (Intersection Observer API, event delegation, mouse tracking)
- **Icons**: [Lucide Icons](https://lucide.dev/) (lightweight SVG icon library)
- **Fonts**: [Outfit](https://fonts.google.com/specimen/Outfit) + [Fira Code](https://fonts.google.com/specimen/Fira+Code) from Google Fonts

## 🚀 Running Locally

No build step needed — just open the file:

```bash
git clone https://github.com/fazal-khn/portfolio-website.git
cd portfolio-website
open index.html
```

Or serve it with any static file server:

```bash
npx serve .
# → http://localhost:3000
```
